# Contract

## Setup

```bash
$ npm install
```

## Compile

```bash
$ npx hardhat compile
```

## Deploy (Testnet)

- ShareKey.sol

```bash
$ npx hardhat run --network mumbai scripts/shareKey.deploy.ts
```

- FriendConnectShare.sol

```bash
$ npx hardhat run --network mumbai scripts/friendConnectShare.deploy.ts
```
